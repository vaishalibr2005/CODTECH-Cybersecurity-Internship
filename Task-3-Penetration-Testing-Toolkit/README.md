# 🛠 Task 3 – Penetration Testing Toolkit

## 📌 Project Overview

The Penetration Testing Toolkit is a modular Python-based security tool developed as part of the CODTECH Cyber Security Internship.

This toolkit includes multiple modules used in basic penetration testing such as:

- Port Scanner
- Brute Force Module

The tool is designed with a menu-driven interface to make it simple and user-friendly.

---

## 🎯 Objective

The objective of this task is to:

- Build a modular penetration testing toolkit
- Implement multiple security testing tools
- Understand basic network scanning techniques
- Simulate brute-force attack logic
- Organize code into separate modules

---

## 🛠 Technologies Used

- Python
- socket module
- threading (optional)
- sys module

---

## 🔍 Features

### 1️⃣ Port Scanner
- Scans open ports on a target system
- Identifies active services
- Helps in network reconnaissance

### 2️⃣ Brute Force Module
- Simulates password guessing attempts
- Demonstrates brute-force attack logic
- Educational demonstration of password security risks

## ▶️ How to Run the Program

1. Open Command Prompt or Terminal.
2. Navigate to the project folder:

   cd Task-3-Penetration-Testing-Toolkit

3. Run the main file:

   python main.py

4. Choose an option from the menu:
   - 1 → Port Scanner
   - 2 → Brute Force Module
   - 3 → Exit
