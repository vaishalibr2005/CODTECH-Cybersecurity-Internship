# 🔒 Task 4 – Advanced Encryption Tool (AES-256)

## 📌 Project Overview

The Advanced Encryption Tool is a Python-based application developed as part of the CODTECH Cyber Security Internship.

This tool encrypts and decrypts files using the AES-256 (Advanced Encryption Standard) algorithm to ensure strong data protection and confidentiality.

It provides a simple and user-friendly command-line interface for secure file encryption and decryption.

## 🔐 What is AES-256?

AES (Advanced Encryption Standard) is a widely used symmetric encryption algorithm.

AES-256:
- Uses a 256-bit key
- Provides very high security
- Used in banking systems, government systems, and secure communications

---

## ▶️ How to Run the Program

1. Install required library:

   pip install cryptography

2. Open Command Prompt or Terminal.
3. Navigate to the project folder:

   cd Task-4-Advanced-Encryption-Tool

4. Run the program:

   python encryption_tool.py

5. Choose:
   - 1 → Encrypt File
   - 2 → Decrypt File

6. Enter the file name and key when prompted.
